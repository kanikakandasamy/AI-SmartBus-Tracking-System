# AI Smart Bus Tracking and Notification System

## Project Title

**AI Smart Bus Tracking and Notification System for College Transportation**

---

## Project Overview

The **AI Smart Bus Tracking and Notification System** is a web-based application designed to improve the efficiency of college transportation. The system enables students to track the live location of college buses, receive real-time notifications, and view the estimated time of arrival (ETA). It also provides administrators with tools to manage buses, routes, drivers, and student information effectively.

By integrating GPS-based tracking and intelligent notifications, the system helps reduce waiting time, enhances communication between students and transport administrators, and ensures a more reliable transportation experience.

---

## Key Features

- Real-time bus location tracking
- Estimated Time of Arrival (ETA)
- Instant notifications and alerts
- Student, Driver, and Admin dashboards
- Bus and route management
- Secure user authentication
